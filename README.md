# NLP Topic Analysis — Consumer Complaint Dataset

A Python NLP pipeline that extracts the most prevalent topics from consumer complaint texts using **Latent Dirichlet Allocation (LDA)** and **Non-negative Matrix Factorization (NMF)**. Built as part of a data science course project simulating a municipal complaint analysis system.

---

## Project Structure

```
nlp-topic-analysis/
├── src/
│   ├── preprocess.py       # Text cleaning, tokenization, lemmatization
│   ├── vectorize.py        # BoW and TF-IDF vectorization
│   ├── lda.py              # LDA topic modelling with coherence scoring
│   └── nmf.py              # NMF topic modelling with coherence scoring
├── results/
│   ├── lda_topics.txt          # LDA topic output
│   ├── nmf_topics.txt          # NMF topic output
│   ├── lda_coherence_plot.png  # LDA coherence curve
│   └── nmf_coherence_plot.png  # NMF coherence curve
├── .gitignore
├── requirements.txt
└── README.md
```

## Dataset

🔗 https://www.kaggle.com/datasets/selener/consumer-complaint-database

Download the dataset and place it as:
```
data/data.csv
```

> The `data/` and `models/` directories are excluded from this repository due to file size. All artifacts are regenerated by running the pipeline.

## Setup

### 1. Clone the repository
```bash
git clone https://github.com/TumisangF/nlp-topic-analysis.git
cd nlp-topic-analysis
```

### 2. Create a virtual environment
```bash
python -m venv nlp_env
```

Activate it:
- **Windows:** `nlp_env\Scripts\activate`
- **Mac/Linux:** `source nlp_env/bin/activate`

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

---

## Running the Pipeline

Run the scripts in order from the `src/` directory:

```bash
cd src

# Step 1: Preprocess raw text
python preprocess.py

# Step 2: Vectorize cleaned text (BoW + TF-IDF)
python vectorize.py

# Step 3: Extract topics with LDA
python lda.py

# Step 4: Extract topics with NMF
python nmf.py
```

> **Note:** `lda.py` and `nmf.py` run a coherence search over 5 topic counts each (5, 10, 15, 20, 25) and select the optimal number automatically. On large datasets this may take 30–60 minutes per script.

## Results Summary

| Model | Input      | Optimal Topics | Best Coherence (c_v) |
|-------|------------|----------------|----------------------|
| LDA   | BoW        | 25             | 0.5473               |
| NMF   | TF-IDF     | 15             | 0.6120               |

NMF produced sharper, more interpretable topics with a higher coherence score. Key themes identified include student loans, mortgage foreclosure, debt collection, identity theft, unauthorized credit inquiries, and auto finance.

## Dependencies

Key libraries used:

- `pandas` — data loading and manipulation
- `nltk` — stopword removal, tokenization
- `scikit-learn` — CountVectorizer, TfidfVectorizer, LDA, NMF
- `gensim` — coherence scoring (CoherenceModel, c_v)
- `scipy` — sparse matrix storage
- `matplotlib` — coherence curve plots

Full dependency list: `requirements.txt`
